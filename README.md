# 🏔️ Fondation Tighremt

Site web officiel de la Fondation Tighremt pour le développement durable du Haut Atlas marocain.

## 🌐 Site Live

**URL** : https://wasslamenace.github.io/fondation-tighremt/

---

## 📋 Projet

**Mission** : Soutenir le développement économique et social des communautés rurales du Haut Atlas (Vallée du Dadès, Tinghir/Boumalne Dadès).

**Engagement** : Redistribution de 1% des revenus du projet TIGREMT (énergie solaire + agrivoltaïsme) vers des associations locales partenaires.
